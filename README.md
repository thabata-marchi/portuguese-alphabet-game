# ABC Safari

![ABC Safari Demo](docs/demo/abc-safari.gif)

Educational game focused on **learning the Portuguese language** and early literacy.  
It combines **reinforcement learning** and **voice recognition** to teach the alphabet in an interactive and fun safari-themed environment.

## About the Project

**ABC Safari** is a web-based educational game designed to support **Portuguese language learning** for children.  
Players learn letters of the alphabet by interacting with safari characters and environments.

The game uses **artificial intelligence** to dynamically adapt the difficulty of the activities and **voice recognition** to validate the pronunciation of letters, helping children develop reading, speaking, and visual association skills.

### Technologies

- **[PixiJS](https://pixijs.com/)** — 2D rendering and animations
- **[TensorFlow.js](https://www.tensorflow.org/js)** — Reinforcement learning and voice recognition
- **[GSAP](https://gsap.com/)** — Animations and transitions
- **[Howler.js](https://howlerjs.com/)** — Sound effects and audio
- **[Webpack](https://webpack.js.org/)** — Bundling and development server

### Features

- Voice recognition to identify letters spoken by the child
- Reinforcement learning agent that adapts the difficulty level
- Learning the **Portuguese alphabet**
- Animated characters and an interactive mascot
- Sound effects and visual feedback
- Child-friendly and intuitive interface
